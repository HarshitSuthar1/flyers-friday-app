# Screenshots Directory

This folder contains app screenshots for the README and submission.

## Required Screenshots:

1. **feed_screen.png** - Home screen showing flyer list and categories
2. **flyer_details.png** - Detail view of a selected flyer
3. **create_posting.png** - Form for creating new postings

## How to Capture Screenshots:

### Using Android Studio Emulator:
1. Run `flutter run` to launch the app
2. Click the camera icon on the emulator toolbar
3. Save screenshots here

### Using Windows Snipping Tool:
1. Press `Windows + Shift + S`
2. Select area to capture
3. Save to this folder

### Using Flutter DevTools:
1. Run app with `flutter run`
2. Press `s` in terminal to take screenshot
3. Screenshots saved to project root, move them here

## Naming Convention:
- Use descriptive names
- Use lowercase with underscores
- Format: PNG (preferred) or JPG

## Example:
```
screenshots/
├── feed_screen.png
├── flyer_details.png
├── create_posting.png
└── (optional) demo.gif
```
